# Contributing

This is a teaching repository where instructor notebooks generated from my courses will end up here.

## Types of Contributions

We accept contributions such as:

- Improved explanations
- Corrected code examples
- Additional exercises
- Clarifications and fixes to existing materials

## How to Contribute

1. Fork the repository
2. Create a branch for your changes
3. Submit a pull request with a clear description of your changes

## Notes

- All contributions will be made available under the same license as the original repository
- This is an educational repository, so educational materials are preferred
